About Us

Our hospital appointment management system, SHAMS (Simplified Hospital Appointments Management System), revolutionizes healthcare accessibility by providing a seamless platform for scheduling medical appointments. With SHAMS, patients can easily book appointments for a wide range of medical services, from dental care to ophthalmology, through a user-friendly interface. The system intelligently assigns appropriate doctors based on the selected service, ensuring patients receive specialized care tailored to their needs. Additionally, doctors benefit from efficient schedule management tools, allowing them to plan their appointments, block off time for breaks, and reschedule as necessary. Administrators have comprehensive oversight, managing user roles, monitoring system performance, and ensuring smooth operation. SHAMS streamlines the appointment process, reduces wait times, and enhances the overall efficiency of healthcare delivery.

User Story 1: Booking an Appointment with Service Selection

As a: Patient
Name: Mr. Murenzi Andrew
I want to: book an appointment for a specific medical service
So that: I can receive specialized medical care

Scenario:

  1. Mr. Murenzi Andrew logs into the hospital appointment system using his credentials Through Hospital website.
  2. He navigates to the "Request Appointment" section.
  3. He gets a form to put patient information.
  4. He selects the desired service from a list of options (e.g., Dental, Internal Medicine, Gynecology, Ophthalmology).
  5. Upon selecting "Dental," the system automatically assigns Dr. UWINEZA Ange, the Dental Surgeon.
  6. He chooses an available date and time slot for his appointment.
  7. He confirms the appointment booking.
  8. Mr. Murenzi Andrew receives a confirmation notification via email and SMS.

Acceptance Criteria:

  • The system displays a list of medical services.
  • The system assigns the correct doctor based on the selected service.
  • The system successfully books the appointment and sends confirmation notifications.
  • The appointment appears in Mr. Murenzi Andrew's appointment list.

User Story 3: Administering the System

As a: Admin
Name: Technician Ngoga Gerald
I want to: oversee the appointment system and manage user roles
So that: I can ensure the system runs smoothly and all users have appropriate access

Scenario:

  1. Technician Ngoga Gerald logs into the hospital appointment system using his admin credentials.
  2. He navigates to the "Admin Dashboard" section.
  3. He manages user accounts, creating new doctor and patient accounts as needed.

  5. He monitors system performance and resolves any issues.

Acceptance Criteria:

  • The system provides a comprehensive admin dashboard with system activity summaries.
  • The system allows Technician Ngoga Gerald to manage user accounts and roles effectively.
  • The system includes tools for monitoring performance and addressing issues.
